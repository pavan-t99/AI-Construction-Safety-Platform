Day 1 — OpenCV Basics

You should be able to explain:

What is a frame?
A frame is a single image from a video stream.

Example:

30 FPS
↓
30 frames every second
What does cap.read() return?
ret, frame = cap.read()

ret:
True  -> frame captured successfully
False -> frame not captured

frame:
The actual image (NumPy array)
Why is a loop needed?

Because:

Video = many frames

Need:->while True:

to process every frame.

Why waitKey()?

Allows OpenCV to:

Show window
Capture keyboard input

Example:

if cv2.waitKey(1) == ord('q'):

quit when Q is pressed.

Day 2 — YOLO Basics

You should know:

What is YOLO?
Object Detection Model

Detects:

Person
Car
Helmet
Mask
etc.
What is inference?
Image
↓
Model
↓
Predictions
What is results?
results = model(image)

Stores all detections.

Why results[0]?

Because YOLO returns a list.

Even for one image:

results[0]

is the first image result.

Day 3 — Bounding Boxes

You should know:

What is a bounding box?

Rectangle around an object.

What are
xmin
ymin
xmax
ymax

?

xmin,ymin -> top-left
xmax,ymax -> bottom-right
Why convert to int?

OpenCV drawing functions require:

Pixel coordinates

which must be integers.

What is confidence?

Example:

0.92

means:

92% confidence

You should know:

YOLO label format
class_id

y_center

Normalized coordinates

image_width
image_height

Day 7 — Violation Logic & Evidence

You should know:

What is violation logic?

Example:

if class_id == 8:

means:

NO-Hardhat
↓
HELMET VIOLATION
Why use a violation list?
violation_list = [6,7,8,9,10]

Makes detection easier.

What is evidence capture?
cv2.imwrite(...)

Saving proof of a violation.

Why use violation_found?

Without it:

3 violations
↓
save image 3 times

With it:

3 violations
↓
save image once
